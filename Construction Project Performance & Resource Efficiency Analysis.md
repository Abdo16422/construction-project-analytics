#  Construction Project Performance & Resource Efficiency Analytics


##  Project Overview

This project presents a **data-driven analysis of construction project performance**, aiming to uncover operational inefficiencies and improve decision-making using real-world KPIs.

The analysis focuses on understanding how **cost, resources, risk, duration, and equipment usage** impact overall project performance.

The goal is to transform raw project data into **actionable business insights** that can optimize construction operations and resource allocation.

---

##  Business Objectives

- Evaluate financial performance across construction projects
- Measure resource allocation efficiency and utilization
- Identify high-risk vs low-risk project patterns
- Analyze project duration variations and delays
- Assess equipment usage efficiency and bottlenecks
- Support data-driven decision-making in project management

---

##  Key Performance Indicators (KPIs)

-  Average Project Cost
-  Average Risk Level
-  Average Project Duration
-  Resource Efficiency Score
-  Equipment Utilization Rate

---

##  Analytical Approach

### 1. Data Understanding
- Dataset structure exploration
- Identifying numerical & categorical features
- Checking missing values and data quality

### 2. Exploratory Data Analysis (EDA)
- Distribution analysis of key variables
- Correlation analysis between features
- Group-based comparisons across projects

### 3. KPI Engineering
- Building performance metrics
- Aggregating insights per project type
- Measuring efficiency and utilization rates

### 4. Insight Generation
- Identifying inefficiencies in resource usage
- Detecting cost and risk drivers
- Highlighting optimization opportunities

---

##  Key Insights

- Some projects show **high cost but low efficiency**
- Strong relationship between **project duration and risk level**
- Equipment utilization varies significantly across projects
- Resource efficiency strongly impacts total project cost
